# ПАРАЛЛЕЛЬНЫЕ АЛГОРИТМЫ УМНОЖЕНИЯ МАТРИЦ
Реализация 3 алгоритмов параллельного умножения матриц:
- алгоритм умножения матриц при ленточной схеме разделения данных
- алгоритм Фокса умножения матриц при блочном разделении данных
- алгоритм Кэннона умножения матриц при блочном разделении данных

Разработка и тестирование программы производились на операционной системе Windows 11 с использованием среды разработки Visual Studio Code и 
компиляции через WSL.
